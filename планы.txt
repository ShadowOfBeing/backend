Плюсы сайта на Github Pages:
1) Простейший деплой через git push
2) Полностью открытый open source проект
3) Доступ к данным БД быстрее, чем у классической БД

Минусы сайта на Github Pages:
1) Отсутствует авторизация и скрытый контент (за исключением сложных схем, где контент прячется на облачных сервисах)
2) Громоздкая структура БД
3) Пользователи не могут изменять контент на сайте, за исключением сложных схем с облачными БД
4) Язык разработки - только JavaScript
5) Отсутствуют шаблоны из коробки, например нет готового шаблонизатора для рендеринга страниц
